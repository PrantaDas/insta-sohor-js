Question-2:What does JS event loop do?

Answer:When JavaScript works it monitors two things-stack and queue.First of all,it executes all the functionalities presents in a stack and then take action from the queue.But if the stack remains empty it directly goes to the queue and take the first function from it to execute.This eteration is to be continued untill the stack or queue becomes empty.It's just like calling the function again one after another.
